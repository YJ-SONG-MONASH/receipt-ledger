# 票据账本 Receipt Ledger

一个离线可用的本地记账网页，用于记录项目支出、个人支出、报销状态、待支付状态和发票附件。

作者：Monash University · Ying-Jian Song

## 使用方式

直接双击打开 `index.html` 即可使用，不需要联网或服务器。

推荐使用 Chrome 或 Edge，以获得目录绑定和本地 JSON 自动保存能力。

## 主要功能

- 新增、编辑、删除支出记录
- 按项目查看项目支出
- 按支付人查看个人支出
- 上传 PDF/PNG/JPG 等发票附件
- 自动显示是否开票：有附件为已开票，无附件为未开票
- 批量导入 PDF/PNG，并按文件名自动解析账目
- 导出/导入完整 JSON 数据
- 绑定 `data.json` 后自动同步保存到账本目录
- 表格列宽支持拖拽调整，并保存在本地

## PDF/PNG 批量导入格式

文件名格式：

```text
机构名-支出明细-金额-日期.pdf
```

示例：

```text
北京大学-会员费用-550-2026.04.13.pdf
清华大学-资料费-1200.50-2026.04.14.png
```

导入前必须先在左侧填写或选择“所属项目”。如果左侧已经填写支出明细、支付人、金额、个人/机构、报销状态或待支付状态，会优先使用左侧字段。

## 数据安全

真实数据默认保存在浏览器 IndexedDB 中。点击“绑定data.json”并选择本项目目录后，页面会自动同步保存到本地 `data.json`。

不要把真实的 `data.json`、导出的账本 JSON 或发票文件提交到公开仓库。仓库中只保留 `data.example.json` 作为空示例模板。
